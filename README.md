# Simulador Incoterms 2020

Este é um simulador interativo dos Incoterms 2020, desenvolvido em Streamlit.

## Como executar

1. Instale as dependências:
   ```bash
   pip install streamlit
   ```

2. Execute o app:
   ```bash
   streamlit run PrimeiroCodigo.py
   ```

## Sobre os Incoterms

Os Incoterms definem as responsabilidades de vendedores e compradores em transações internacionais.
